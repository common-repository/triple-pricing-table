== Description ==

<a href="http://xn--diseowebmadrid-tnb.com/tiendas-online/">Live Demo in online shop!</a>

Triple pricing table is a free, responsive and customizable Wordpress table for your site. The easiest way to announce your services and prices with a clean style and very easy installation process.

As well, if you have an eCommerce shop, you can always use as much tables as you want to insert your prefered Wordpress themes.


With the triple pricing table plugin you have no excuse to not advertise the features and prices of your service in an easy way.

Simple, friendly and striking design to focus the attention of your potential users.


This plugin allows you to:

<ul>
<li>Create informative tables</li>
<li>Show your services and prices to your customers</li>
<li>Publish your desired information</li>
<li>Customize colors</li>
<li>Customize fonts</li>
<li>Select all the characters that you want to show. Just fill what you need.</li>
</ul>
After creating your first pricing table, (WP-admin->Settings->Triple Pricing Slider) you can load the plugin using this code:

`[pricingtable id=1] For the table 1`

You can place as many tables as you want, copy the shortcodes, change "id=1" for the provided id and then publish it!


== Installation ==

This section describes how to install the plugin and get it working in less than 5 minutes!

1. Download the pricing plugin for Wordpress
2. Login to your Wordpress administration panel
3. Install the plugin via plugins menu
4. Once downloaded and installed you'll see the activation menu
5. You'll see the new "Triple Pricing Table" option in "Settings"
6. Access to it, fill all the fields and save it
7. That's all!
